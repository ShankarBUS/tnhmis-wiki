An unofficial guide designed for Doctors and Interns to gain basic usage knowledge of TN HMIS.

Wiki pages are generated dynamically from markdown files in `src/docs/*.md`.
To add markdown files to the wiki, include their information in the `src/data/docs-map.json`

> [!NOTE]
> The TN HMIS icon included in this repository is a recreated version of the official icon (16x16).
> It was recreated from scratch and not taken from the official sources.
