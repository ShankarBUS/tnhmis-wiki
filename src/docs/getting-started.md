# Login Page
![Login Page](/src/assets/images/login-page.png)

Each users are assigned a unique username which could be their name or their phone number. Based on their designation, module accessibility will vary.

For example: doctors can only access Doctor's Desk (OPD/IPD/EMR/MIS, etc)

# Home Page
![Home Page](/src/assets/images/home-page.png)

Home page is structured as following:
- Main menu which lists all the **available modules** for the user.
- Tab view presents each service as a tab. All the services are presented within a single main page. Clicking on a service will open it up in a tab under the main page.
